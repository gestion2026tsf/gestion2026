Conectar tu Excel de OneDrive

Necesitas el ID de tu archivo Excel y el nombre de la hoja para jalar la data automáticamente.

1
Abre tu Excel en OneDrive / SharePoint

Ve a OneDrive, abre el archivo Gestión_2026 en Excel Online.

2
Copia el File ID de la URL

En la barra de dirección busca el parámetro resid= o el ID largo. Ejemplo:

https://onedrive.live.com/edit?resid=ABC123XYZ!456

El ID sería: ABC123XYZ!456

3
Para cuenta corporativa (Microsoft 365)

Ve a SharePoint o Teams, busca el archivo, clic en "..." → Detalles → copia el ID de elemento.

O abre en Excel Online y la URL tendrá: /Items/ID_AQUI/workbook
